# Contribution Guidelines

Thanks for helping keep **Awesome Context AI** awesome.

## What belongs here

Tools, frameworks, models, and resources for building AI with rich context from screen recordings, audio, activity tracking, and personal data. If it helps an AI understand what someone has **seen, heard, said, or done**, it fits.

## How to contribute

1. Fork this repo and create a branch.
2. Add your entry to the most relevant section, keeping the existing alphabetical-ish / by-relevance order within tables.
3. Open a pull request with a short note on why the entry belongs.

## Quality bar

- **It must work.** Link to the canonical repo or homepage and verify the link resolves.
- **No dead or deprecated projects.** If a project is archived or unmaintained, it gets removed, not added.
- **One line, factual.** Descriptions are concise and avoid marketing language and superlatives.
- **No fabricated facts.** Star counts come from the live shields.io badge; don't invent acquisition, funding, or benchmark claims.
- **Prefer open source.** Commercial tools are welcome when they're category-defining, but lead with open-source options.

## Format

Open-source entries use a table row with a live stars badge:

```markdown
| [Name](https://github.com/owner/repo) | One-line description | ![GitHub stars](https://img.shields.io/github/stars/owner/repo) |
```

Closed-source or model entries use `N/A` in the stars column.
